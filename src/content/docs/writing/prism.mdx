## Prism (OpenAI, 2026)

Prism 是 OpenAI 推出的 **LaTeX 原生科研写作工作区**，发布于 2026年1月27日。

---

## 核心特性

| 特性 | 说明 |
|------|------|
| **LaTeX 原生工作区** | 无缝迁移现有项目 |
| **GPT-5.2 集成** | 强大的 AI 写作辅助 |
| **实时协作** | 多人同时编辑 |
| **中英文支持** | 双语写作无障碍 |
| **公式和图表创建** | 内置可视化工具 |
| **引用管理** | 智能文献引用 |

---

## 与 Overleaf 对比

| 特性 | Prism | Overleaf |
|------|-------|----------|
| AI 集成 | GPT-5.2 原生 | 需外部插件 |
| 实时协作 | ✓ | ✓ |
| 中文支持 | ✓ | 需配置 |
| 价格 | 免费（个人账户） | 有限制 |

---

## 使用场景

### 1. 从头开始写论文

Prism 的 AI 可以帮助：
- 生成论文大纲
- 扩展段落内容
- 优化语言表达
- 检查引用格式

### 2. 协作写作

- 多人实时编辑
- 版本历史记录
- 评论和讨论
- 任务分配

### 3. 中英文写作

- 无缝切换语言
- 智能翻译建议
- 中文学术写作优化
