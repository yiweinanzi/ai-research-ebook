## PaperBanana (arXiv:2601.23265, 2026)

### 论文信息

**标题**: PaperBanana: Automating Academic Illustration for AI Scientists

### 核心功能

- 自动识别论文中的插图需求
- 生成符合顶会风格的初版草图
- 支持多种图表类型的智能推荐

## GPT 辅助流程

### 1. 生成元素清单

让 GPT 根据论文内容生成：

- "图的元素清单"（模块/箭头/标注）
- "布局草案"（从左到右还是从上到下）
- "提示词"（给绘图工具出初版草图）

### 2. 生成提示词模板

```
你是一位专业的学术插图设计师。请为以下论文内容设计一张架构图：

[论文内容/方法描述]

请输出：
1. 图表元素清单（所有模块、箭头、标注）
2. 布局建议（从左到右/从上到下/中心辐射）
3. 每个元素的视觉建议（颜色、形状、大小）
4. 给绘图工具的具体指令
```

### 3. 整合与发布

最后用 PPT/Figma 统一绘制与整合
